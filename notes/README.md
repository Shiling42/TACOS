# Demo Output Figures

This directory contains output figures generated by the demonstration scripts.

## Demo Figures

| File | Description |
|------|-------------|
| `demo_self_assembly.png` | Fuel-driven self-assembly: affinity and concentration bounds with simulation validation |
| `demo_schlogl.png` | Schl√∂gl model: saddle-node bifurcation diagram within thermodynamic bounds |
| `demo_chiral.png` | Frank model: pitchfork bifurcation showing chiral symmetry breaking limits |
| `demo_efm_methods.png` | EFM enumeration: computation time and scaling comparison |

## Regenerating Figures

```bash
cd code/crn
python demos/demo_01_self_assembly.py
python demos/demo_02_schlogl.py
python demos/demo_03_chiral.py
python demos/demo_04_efm_methods.py
```

Add `--svg` flag to also generate SVG versions.
